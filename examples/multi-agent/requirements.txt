# -----------------------------------------------------------------------------
# Version Pinning Strategy
# -----------------------------------------------------------------------------
#
# We pin exact versions (==) instead of ranges (>=) to ensure:
# 1. Reproducibility: Same code â†’ same behavior
# 2. No surprise breaking changes
# 3. Easier debugging (everyone uses same versions)
#
# For AI/ML Scientists:
# This is like fixing random seeds - ensures your experiments are reproducible.
# Trade-off: Miss automatic bug fixes, must manually update versions.
#
# Alternative (more flexible but riskier):
# langgraph>=0.0.26,<0.1.0  # Allow patch updates, not major/minor
#
# -----------------------------------------------------------------------------
# Dependency Size Notes
# -----------------------------------------------------------------------------
#
# Approximate installed sizes:
# - boto3 + botocore: ~60MB
# - sagemaker: ~15MB
# - langchain + langgraph: ~25MB
# - tavily-python: ~1MB
